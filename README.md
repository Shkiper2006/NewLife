# Жизнь с Нуля — MVP вертикальный срез

Играбельный браузерный прототип survival-adventure в современном городе после техногенной катастрофы. Текущая версия — это **MVP-срез стадии “младенец”** с локальным кооперативом через WebSocket.

## Что уже есть в прототипе

- **Vite + React + TypeScript** как базовый фронтенд-стек.
- **Three.js + @react-three/fiber + @react-three/drei** для 3D-сцены.
- **Rapier physics** для базовой физики игрока и окружения.
- **Zustand** для состояния игры и HUD.
- **Node.js + WebSocket** для простой кооперативной синхронизации двух и более игроков.
- Локация **“роддом”** с ощущением большого мира за счёт масштаба младенца.
- Игрок в стадии **младенца** с перемещением **ползком** и **follow-camera**.
- Подбор предметов: **тряпки**, **бутылочки с молоком**, **простой мусор/материалы**.
- Инвентарь, голод, жажда, здоровье.
- Крафт:
  - `тряпки -> верёвка`
  - `верёвка + материалы -> простое укрытие`
- Размещение укрытия в мире.
- Ранняя угроза: **крыса**, патрулирующая коридор выхода.
- Цель уровня: **открыть путь наружу из роддома** и доползти до пролома.
- Подготовленная data-driven основа для будущих стадий:
  - ребёнок: ходьба, лазание, простые инструменты;
  - подросток: бой, оружие, база, ловушки;
  - взрослый: инженерия, инфраструктура, сюжетный финальный выбор.

## Управление

- `WASD` / стрелки — ползти
- `E` — подобрать ближайший предмет
- `1` — скрафтить верёвку
- `2` — скрафтить простое укрытие
- `Q` — поставить укрытие рядом с младенцем
- `F` — выпить бутылочку молока

## Как играть

1. Найдите в палате и рядом с вентиляцией **2 тряпки**.
2. Нажмите `1`, чтобы сделать **верёвку**.
3. Найдите **2 единицы мусора/материалов**.
4. Нажмите `2`, чтобы сделать **простое укрытие**.
5. Подползите к **жёлтому кругу** у пролома наружу.
6. Нажмите `Q`, чтобы положить укрытие на осколки и открыть безопасный маршрут.
7. Избегайте крысы и доползите до **голубой метки** у выхода.

## Локальный кооператив

1. Запустите проект командой ниже.
2. Откройте игру в двух вкладках браузера.
3. Обе вкладки подключатся к локальному WebSocket-серверу `ws://localhost:2567`.
4. Игроки синхронизируют позиции, подбор предметов и установку укрытия.

## Запуск

```bash
npm install
npm run dev
```

После запуска:
- клиент Vite: `http://localhost:5173`
- кооп-сервер: `ws://localhost:2567`

## Проверка типов и production-сборка

```bash
npm run check

```

## Сюжетный каркас на будущее

Город был искусственно изолирован после техногенной катастрофы ради защиты детей. В полной версии взросление должно вести игроков к центру управления городом, где появится финальный выбор:

- открыть город внешнему миру;
- сохранить изоляцию и строить автономное общество.

## Примечания

- Некоторые игровые системы пока реализованы как **MVP-заглушки**: например, AI крысы, логика выхода и кооп-авторитет сервера intentionally упрощены для вертикального среза.
- Архитектура оставлена расширяемой, чтобы следующие возрастные стадии добавлялись через data-driven конфиги, а не через один монолитный файл.
